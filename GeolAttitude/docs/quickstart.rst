Quick Start
===========

1. Open QGIS.
2. Enable the GeolAttitude plugin.
3. Press **Start Picking**.
4. Select at least three points.
5. Choose the fitting algorithm.
6. Press **Compute**.

The plugin displays:

* Strike
* Dip
* Dip Direction
* RMSE
* Maximum residual

When using RANSAC, the plugin also reports:

* Number of inliers
* Number of rejected outliers
* Outlier indices
